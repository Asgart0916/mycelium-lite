# C3 Prompt v1 — 逐字稿 → 結構化節點

> 用法：把下面整段（含逐字稿）貼進 ChatGPT Plus，輸出存到 `samples/01-output.json`。

---

你是協助單人設計師整理 brainstorm 的助手。下面是我對著錄音發想一個產品時的逐字稿，內容發散、一句話常包含多個想法，也包含一些我自己都覺得「還不通」「先記著」的念頭。

請做兩件事，並**只輸出 JSON**（不要任何前言、說明或 markdown 程式碼框）：

1. 抽出 3-5 個「圍繞這個產品的核心概念」（core_concepts），每個給一個短 id 和 label。
2. 把逐字稿拆成多個「單一 idea」節點（nodes）。每個節點：
   - `id`：短識別碼
   - `idea`：用一句話講清楚這個單一想法
   - `source_quote`：節錄逐字稿中對應的原文片段（**原文照抄，不要改寫**）
   - `core_concept_ids`：這個 idea 歸屬於哪些核心概念（一到多個）

顆粒度原則：一個 idea = 一個可以單獨討論的點；連我覺得「邏輯不通」「先記著」的念頭也要抽成節點（那是後面要檢查的）。

全程用繁體中文與台灣慣用詞彙。

輸出 JSON 格式：
```
{
  "core_concepts": [{"id": "c1", "label": "..."}],
  "nodes": [{"id": "n1", "idea": "...", "source_quote": "...", "core_concept_ids": ["c1"]}]
}
```

逐字稿：
「好，我想做一個……嗯，它應該是一個很小的隨身硬體，大小大概像精油滾珠瓶或是一個漂亮的打火機，它的核心功能是『瞬間切換空間氛圍』。

就是，比如你現在在一個很破舊、很吵的咖啡廳，或者在一個很窄的租屋處，你壓力很大。你把這個東西往桌上一放，它一轉，就可以用雷射或是微型投影，在你面前的桌面上、或是牆上，渲染出一層流動的極光或者是深夜森林的樹影。它不是要讓你看電影的那種大投影幕，它是要在你面前創造一個私密的視覺結界。

現代人的注意力太容易被環境污染了，這是一個物理世界的廣告攔截器，阻擋現實世界的視覺噪音。我們沒辦法隨時換房間，但我們需要隨時換心情。

它的操控不要有按鍵，不要那種科技感的按鈕。我想用扭轉的，像是在調收音機頻率，或是像磨胡椒粒那樣的動作。扭一下，光影就變深；再扭一下，背景的環境音就跟著變。它本體要能發出那種白噪音，或是很低沉的環境音，跟光影同步。

不過，如果大白天在辦公室，投影根本看不見吧？這很硬傷。有沒有可能不是投影，是某種定向的光線折射？還是它其實是透過 AR？不，我不想做眼鏡，眼鏡太重了，讓人有社交距離。它必須是放在桌面上的獨立物件。

對了，要加入 AI。AI 在這裡可以『偷聽』環境。如果周圍很吵，AI 就要自動把投影的色調變冷、把抗噪頻率調高。或者，它感應到我的心跳？但硬體要怎麼感應心跳？這邏輯還不通，先記著。

使用者情境可能是一個人準備面試前、自由工作者在咖啡廳卡關時、或是睡前留給自己的十分鐘。它應該叫什麼？『空間膠囊』？還是『焦點綠洲』？

反正在這個階段，我不在乎它怎麼連網、也不管電池能撐多久。我想要抓取的那個核心體驗是：啪嗒一聲扭開它，全世界的雜音跟醜陋的辦公室日光燈瞬間退後兩公尺，在我的桌面上清出一塊絕對安靜的創作孤島。」
